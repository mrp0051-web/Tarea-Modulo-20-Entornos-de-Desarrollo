
# Calculator Project

## Project Description

This project is a simple web calculator developed using HTML, CSS, and JavaScript.  
The calculator can perform basic arithmetic operations:

- Addition
- Subtraction
- Multiplication
- Division

A clear button feature was later added during the branch development phase.

-----------------------------------------------------------------------------------------------------------

# Development Process

## Step 1 - Project Setup

First, a local project folder called `calculator-project` was created and opened in Visual Studio Code.

The following files were created:

- index.html
- style.css
- script.js
- README.md

-----------------------------------------------------------------------------------------------------------

## Step 2 - Building the Calculator

The calculator interface was created using HTML.

CSS was used to style the application and improve the visual appearance.

JavaScript functions were implemented to handle:

- Addition
- Subtraction
- Multiplication
- Division

A validation for division by zero was also added.
